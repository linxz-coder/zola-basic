# maxlength属性

默认是140个字符。

一般设置为`-1`，指用户可以无限输入。

# confirm-type属性

可以改变键盘确认键。

`send`， `next`，`search`，`go`，`done`


# 按钮button知识

`plain`属性有镂空效果。
