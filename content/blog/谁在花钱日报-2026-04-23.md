今天，资金继续流向“电网基建、头部AI联盟、消费升级品类”。

# 政府

国家电网一季度固定资产投资超过1290亿元，同比增长37%，并带动上下游投资超过2500亿元。特高压与电网补强持续推进，电力基础设施链条景气度维持高位。

# 国际政府与开发资金

美国对沙特F-15后勤保障潜在军售获批，金额30亿美元。防务维护类采购仍在扩张，军工后勤与长期服务能力价值提升。

# 投资机构

Amazon与Anthropic宣布扩大合作：Amazon先投50亿美元，未来可再投至多200亿美元；Anthropic承诺未来10年在AWS支出超1000亿美元。AI头部公司与云平台的绑定进一步加深。

# 企业

OpenAI宣布完成1220亿美元融资，投后估值8520亿美元。AI基础设施与平台化竞争进入更高资金密度阶段。

国内推理GPU厂商曦望完成超10亿元融资，累计融资约40亿元。资金将用于推理GPU量产交付和软件生态建设，国内AI算力硬件端继续加速。

# 富豪资本

加密富豪Justin Sun就美国民事欺诈案达成1000万美元和解。富豪资本在高监管行业的资金动作，不只体现在投资，也体现在合规和法律成本支出。

# 平民消费

3月消费资金继续向升级品类集中：

1. 通讯器材类：952亿元，同比增长27.3%
2. 文化办公用品类：407亿元，同比增长21.7%
3. 体育娱乐用品类：121亿元，同比增长25.4%
4. 家具类：151亿元，同比增长18.1%
5. 金银珠宝类：356亿元，同比增长11.7%

同期，3月社零总额41616亿元，同比增长1.7%。消费增量更偏“功能升级与体验升级”品类。

# 高薪职位

当前更值得关注的5个高薪岗位是：

1. AI科学家 / AI负责人：平均月薪约137153元
2. 算法研究员：月薪普遍约70000元
3. 大模型算法工程师：月薪普遍约70000元
4. AIGC算法工程师：月薪普遍约70000元
5. AI岗位整体：平均月薪约60738元

普通人更现实的路径，是先进入AI工程化岗位，再向高阶算法岗位升级。

# 今日结论

今天最关键的数字是：电网投资1290亿元、军售30亿美元、Amazon对Anthropic新增投资50亿美元并可追加200亿美元、Anthropic承诺AWS超1000亿美元、OpenAI融资1220亿美元、曦望融资超10亿元，以及通讯器材消费952亿元（+27.3%）。资金仍在向“可落地、可持续、可形成长期回报”的方向集中。
