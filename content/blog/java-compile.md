要想让Java代码运行起来，需要进行编译。编译步骤：

编写-编译-运行

编写.java -> bin/javac编译成字节码 -> bin/java运行

![java-compile](https://linxz-aliyun.oss-cn-shenzhen.aliyuncs.com/images/javac.exe.png)

# 编译实例代码

## 编译

```bash
javac HelloWorld.java
```

## 运行

```bash
java HelloWorld
```
